## Currículo:

Currículo pessoal desenvolvido em HTML e CSS, com foco em tipografia, hierarquia visual e um toque de identidade ligada à área de tecnologia.

🔗 **Site ao vivo:** https://pamelamachad00.github.io/curriculo/

## Conteúdo:
- Objetivo profissional
- Habilidades técnicas
- Formação acadêmica
- Projetos / portfólio

## Tecnologias:
- HTML5
- CSS3 (arquivo separado, sem estilos inline)
- Google Fonts (Fraunces, IBM Plex Sans, IBM Plex Mono)

## Como rodar localmente:
```bash
git clone https://github.com/PamelaMachad00/curriculo.git
cd curriculo
```
Depois é só abrir o `index.html` no navegador.

## Design:
Layout de coluna única com rótulos de seção em estilo "comentário de código" (`// perfil`, `// habilidades`) e uma linha do tempo inspirada em histórico de commits para formação e projetos — uma referência discreta à área de estudo.
